# Black & Scholes Option Calculator

## Example

```
underlyingPrice - 16929.6
exercisePrice - 17000
daysToExpiry - 7
interest - 10
premium - 109.55
dividendYield - 0

callIV = ImpliedCallvolatility(underlyingPrice,exercisePrice,daysToExpiry,interest,premium,dividendYield)
callIV = 13.62
```
